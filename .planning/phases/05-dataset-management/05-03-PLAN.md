## Trust Boundaries

| Boundary | Description |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-09 | Tampering | column-metadata-grid.tsx | mitigate | AG Grid agSelectCellEditor constrains values to predefined options (string/number/date/currency for type, dimension/measure/time/none for role, etc.). Free-text fields (displayName, formatString) validated by Pydantic on backend. |
| T-05-10 | Tampering | dataset-editor.tsx SQL field | accept | SQL is executed by Superset SQL Lab, not by RecViz directly. Superset applies its own query validation. Auth deferred to SECU-01. |
| T-05-11 | Denial of Service | Run Query button | mitigate | Button shows loading state and is disabled during execution, preventing double-submit. Backend httpx timeout at 120s prevents runaway queries. |
| T-05-12 | Information Disclosure | delete-dataset-dialog.tsx | accept | Shows chart names that reference a dataset -- this is intentional for the dev to know what to fix before deleting. All users are internal dev team. |
</threat_model>
