## Interaction Contract

### Card Click -> Detail Panel
1. User clicks KPI card or row
2. Sheet slides in from right (200ms ease-out)
3. Sheet shows threshold-colored `border-l-2` accent on left edge
4. KPI value animates via CountAnimation (0.8s)
5. Sticky footer shows Edit KPI + Delete actions

### Grid/List View Toggle
1. User clicks toggle group item
2. Current view fades out (opacity 1->0, 100ms)
3. `AnimatePresence mode="wait"` waits for exit
4. New view fades in with stagger (opacity 0->1, y 8->0, per-item 50ms delay)

### Builder Step Navigation
1. User clicks step trigger (if not locked)
2. Previous step content fades out (100ms)
3. New step content fades in (150ms)
4. When step completes (Continue clicked), checkmark scales in (spring)
5. Next step auto-opens

### Builder Preview Updates
1. User changes metric/format/threshold config
2. Preview crossfades (200ms opacity) to new value
3. CountAnimation re-triggers with new number

### Delete Flow
1. User clicks delete icon (detail panel footer or builder header)
2. Dialog opens, checks references via API
3. If no references: shows confirmation with "Delete" destructive button
4. If references exist: shows blocker with dashboard list and "Keep KPI" button
