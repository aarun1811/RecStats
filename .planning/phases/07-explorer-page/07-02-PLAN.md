       - Type `SELECT * FROM RECVIZ_DASHBOARDS` (or another seeded table) into Monaco editor
       - Click Run button (or use Cmd+Enter)

       - Verify grid re-themes correctly — screenshot in light mode, toggle to dark mode, screenshot again
    6. **Large result set (per D-14):**
       - Execute `SELECT * FROM RECVIZ_DATASETS` or a query that returns many rows
       - Verify pagination controls appear (25/50/100 per page selector)
       - Verify grid does not crash
    7. **Error handling (per D-15):**
       - Execute `SELECT * FROM nonexistent_table_xyz`
       - Verify structured error state appears with Oracle error message
       - Verify error state uses destructive color tokens
    8. **Save as Dataset (per D-07, D-08):**
       - Execute a successful query
       - Click "Save as Dataset" button
       - Fill in dataset name (e.g., "Explorer Test Dataset")
       - Save and verify success
       - Navigate to Datasets page and verify the new dataset appears in the list
       - Navigate back to Explorer
    9. **Export CSV (per D-13):**
       - Run a query, click CSV export button
       - Verify download initiates (Playwright can check for download event)
    10. **Copy to clipboard (per D-13):**
        - Run a query, click Copy button
        - Verify toast "Copied to clipboard" appears
    11. **AG Grid theme verification (per D-03, D-04):**
        - Take screenshot in light mode
        - Toggle to dark mode
        - Take screenshot in dark mode
        - Verify grid background and text colors change appropriately
    12. **Empty state verification (per D-10):**
        - Navigate to Explorer fresh (no query run yet)
        - Verify empty state shows with BarChart3 icon and "Run a query to see results" text
    13. **Dark mode full verification (per D-12):**
        - Toggle to dark mode
        - Verify all Explorer panels (schema browser, editor, results) render correctly
        - No white backgrounds, no unreadable text, no broken borders

    Take screenshots at each major step. If any step fails, note the failure and continue with remaining steps.
