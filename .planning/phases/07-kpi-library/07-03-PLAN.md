## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-07 | Denial of Service | Library card queries | accept | Each card queries dataset independently. With large KPI counts, this could be many queries. Mitigated by staleTime (5min cache) and TanStack Query dedup. Acceptable for dev-team usage. |
| T-07-08 | Spoofing | Delete without reference check | mitigate | Delete dialog always calls useKpiReferences before enabling delete button. Backend delete endpoint does not check references (Phase 8 adds this), but UI gate prevents accidental deletion. |
</threat_model>
