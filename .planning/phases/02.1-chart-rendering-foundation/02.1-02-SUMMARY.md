## Decisions Made
- buildSeries exported as a named export for direct unit testing. No signature or logic changes -- just the export keyword. This enables testing the pure column mapping logic without React rendering overhead.
- Each chart type gets a dedicated data source with columns specifically shaped for that type, rather than one generic data source. This ensures each type exercises its unique column mapping path.
- Waterfall seed data uses actual increments (Revenue: +5M, COGS: -2.1M, etc.) matching AG Charts waterfall series behavior which auto-calculates running totals from positive/negative values.
