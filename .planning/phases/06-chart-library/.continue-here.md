## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Object identity in useEffect deps | `state` object as useEffect dependency causes infinite loop — parent setState → child re-render → new object ref → effect fires again | blocking | Use primitive values or JSON.stringify for object deps in useEffect. Never depend on a state object directly. |
| Missing migration | Alembic migration file created but never applied — backend 500s on new table endpoints, CORS headers missing on 500 responses causes browser flickering | blocking | After any plan that creates a migration, verify table exists: `docker exec recviz-postgres psql -U recviz -d superset_meta -c "\dt recviz_*"` |
| Seed data assumes tables | Test datasets referenced tables (transactions, breaks) that don't exist in recon_data — only showcase_* tables exist | advisory | Always verify table existence before writing seed SQL |
